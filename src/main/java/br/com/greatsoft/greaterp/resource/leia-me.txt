            Informações Gerais
            ------------------

Para que o sistema funcione, o servido H2 TCP deve estar inicializado.
Independente de a máquina estar rodando em modo servidor ou cliente.
O arquivo config.xml contem os dados de acesso ao BD.
Quando o sistema é executado pela primeira vez o arquivo cfg.xml é criado
automaticamente. No diálogo que aparecerá as seguintes informações são
necessárias:
host: ip da máquina que será servidor ou localhost se a própria máquina
for servidor;
porta: 9801.
BD: greatsales.
modo: server ou client.         


         Como Iniciar o H2 em modo TCP no startup do Sistem Operacional
         --------------------------------------------------------------

Linux
-----
1 - Copie o arquivo h2.conf para /etc/init (necessario ser root)

Exemplo:

sudo cp /home/username/greatsoft/greatsales/h2/h2.conf /etc/init/h2.conf

2 - Reinicie o Sistema Operacional ou inicie o serviço com o comando:

sudo start h2

Windows
-------
1 - Crie um arquivo chamado h2.bat com o seguinte conteudo:

cd c:\greatsoft\greatsales\h2
java -jar c:\greatsoft\greatsales\h2\greath2server.jar

2 - Salve o arquivo na pasta de inicialização automática do windows.
